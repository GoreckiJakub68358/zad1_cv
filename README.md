Strona internetowa w formacie CV wykonana w czystym **HTML5** z wykorzystaniem **CSS3**.  
Projekt realizowany w ramach zadania zaliczeniowego.

## ✅ Wymagania techniczne

- [x] Poprawny szkielet HTML z atrybutem `lang`
- [x] Semantyczne znaczniki: `header`, `main`, `section`, `footer`
- [x] Dokładnie jedno `h1`
- [x] Sekcja: Kontakt (mailto, tel, miasto)
- [x] Sekcja: O mnie
- [x] Sekcja: Umiejętności (min. 5 pozycji)
- [x] Sekcja: Doświadczenie (min. 2 × `article`)
- [x] Sekcja: Edukacja
- [x] Sekcja: Projekty (min. 2 pozycje)
- [x] Min. 2 linki zewnętrzne z `target="_blank"` i `rel="noopener noreferrer"`
- [x] Zdjęcie z niepustym `alt`
