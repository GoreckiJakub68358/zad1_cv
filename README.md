## ✅ Wymagania techniczne

- [ ] Poprawny szkielet HTML z atrybutem `lang`
- [ ] Semantyczne znaczniki: `header`, `main`, `section`, `footer`
- [ ] Dokładnie jedno `h1`
- [ ] Sekcja: Kontakt (mailto, tel, miasto)
- [ ] Sekcja: O mnie
- [ ] Sekcja: Umiejętności (min. 5 pozycji)
- [ ] Sekcja: Doświadczenie (min. 2 × `article`)
- [ ] Sekcja: Edukacja
- [ ] Sekcja: Projekty (min. 2 pozycje)
- [ ] Min. 2 linki zewnętrzne z `target="_blank"` i `rel="noopener noreferrer"`
- [ ] Zdjęcie z niepustym `alt`
